General notes

Link Ninja is a browser extension (currently only available for Mozilla Firefox) that allows you to select multiple links in one go by drawing a selection rectangle. Once you release the selection, the configured default action will be applied to the links. Link Ninja allows you to set up targeting rules in order to restrict the link elements that are available for selection. Targeting rules can be set up by specifying a URL pattern (ex: *.google.com/*) or CSS selector (ex: a.someclass), and can be configured to include or exclude the results.

Basic usage

By default Link Ninja is set up to start selecting by right clicking and dragging. Upon release of the right mouse button, the configured action will be applied to the selection; which by default is to open the selected links into tabs. If instead of releasing the right button the left mouse button is clicked, a pop-up will be shown with all of the available actions to perform on the selection.

Targeting rules

In order to restrict the selectable links, targeting rules can me specified.

Select "Create targeting rule" from the Link Ninja tool-bar.
