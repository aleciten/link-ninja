What is this?

Link Ninja is a browser extension that allows you to select multiple links in one go by drawing a selection rectangle with your mouse. Once your selection is complete a configurable action can be applied to the result.
Available actions include: open in tabs, open in tabs on demand (ie: load tab contents when clicking the tab), open in tabs deferred (ie: delay between each tab opening), copy to clipboard (allowing custom formatting), and bookmark.

You can set up targeting rules in order to restrict the link elements that are available for selection. Targeting rules can be set up by specifying a URL pattern (ex: *.google.com/*) or CSS selector (ex: .someclass), and can be configured to either include or exclude the results of the match.

Basic usage

By default, Link Ninja is set up to start selecting by right clicking and dragging. Upon release of the right mouse button, the configured action will be applied to the selection; which by default is to open the selected links into tabs. If instead of releasing the right button the left mouse button is clicked, a pop-up with all of the available actions will be presented.

Targeting rules

A link will be available for selection when it matches at least one include rule and no exclude rules.

To create a targeting rule:

  1. Select "Create targeting rule" from the Link Ninja toolbar button.
  2. Set the rule type:
    For Inclusion rules the results of the match will be available for selection.
    For Exclusion rules the results of the match will NOT be available for selection.
  3. Choose the targeting type:
    For CSS selector targeting enter any valid CSS3 selector that will target the intended links. For example: a[target='_blank']
    For URL pattern targeting enter any wildcard pattern that will match the link's target URL. For example: *.imgur.com/*
  4. Choose the URL pattern this rule will be bound to.

To manage configured targeting rules select Manage targeting rules from the Link Ninja toolbar button.
